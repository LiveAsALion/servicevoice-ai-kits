## phxpoolpro.com DNS Review

Alistair showed DNS panel for phxpoolpro.com (Namecheap). Issues identified:

1. **CNAME/A record conflict on www** — both CNAME (→ parkingpage.namecheap.com) and A record (→ 85.239.232.215) exist for www. Delete the CNAME.
2. **URL redirect using HTTP** — @ redirect points to http://www.phxpoolpro.com/ — change to https://
3. **SPF missing Resend** — Resend DKIM is configured but SPF doesn't include spf.resend.com. Fix: `v=spf1 include:spf.efwd.registrar-servers.com include:spf.resend.com ~all`
