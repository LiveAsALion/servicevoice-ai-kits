## Protocol
When Alistair responds to any draft with approval ("good", "send it", "yes", "that works") or rejection ("wrong direction", "try again", "no", "cut it"), Woz writes an entry here immediately.

Over time this log reveals:
- Which output styles land vs miss
- Which task types need more/less autonomy
- Recurring corrections that should become defaults
