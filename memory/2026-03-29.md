## Lessons

- **Cron payloads should never contain hardcoded dates.** Use natural language ("today's date") and resolve at runtime. Hardcoded dates become stale silently — no error fires, the agent just reads the wrong file or falls back to HEARTBEAT_OK.
- **SpeedShield / Civic Speed Monitor rename** — project names drift; project files should have an aliases or formerly-known-as field to avoid confusion across sessions.
