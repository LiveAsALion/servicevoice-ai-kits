## Format Guide
To add a new account:
1. Create Google Workspace account for the project address
2. Enable 2-Step Verification on the account
3. Go to Security → App Passwords → Create → name it "Woz Outreach"
4. Paste the 16-character password next to "App Password:" above
5. Note the scope (what Woz is allowed to send from this address)
