# Memory Consolidation Protocol

Run this during heartbeats every few days (or nightly if active).

## Steps

1. Read `memory/YYYY-MM-DD.md` for the past 2–3 days
2. Identify anything worth keeping long-term:
   - Decisions made
   - Lessons learned
   - New facts about Alistair, projects, tools, or preferences
   - Problems encountered and how they were solved
3. Extract into the right Layer 1 file:
   - New project status → `life/projects/`
   - Preference or style update → `life/areas/operating-principles.md`
   - New tool or platform → `life/resources/tools-and-platforms.md`
   - Completed project → `life/archives/`
4. Update `MEMORY.md` if a summary-level change is warranted
5. Prune daily notes older than 30 days (move to archives if significant, delete if routine)

## Memory Layer Reference

| Layer | Location | Purpose |
|---|---|---|
| 1 — Knowledge graph | `life/` (PARA folders) | Durable, structured, long-term facts |
| 2 — Daily notes | `memory/YYYY-MM-DD.md` | Raw session logs, what happened each day |
| 3 — Tacit knowledge | `life/areas/operating-principles.md` | How Alistair thinks, communicates, operates |
