# Life — Layer 1 Knowledge Graph

This folder is the durable, long-term knowledge base. Structured using PARA:

- **projects/** — Active initiatives with a defined outcome and timeline
- **areas/** — Ongoing responsibilities without a finish line (health, faith, family, finances)
- **resources/** — Reference material, research, tools, and templates
- **archives/** — Completed projects, inactive ideas, old reference material

## How It Works

During daily consolidation (nightly or during heartbeats), Woz reviews the daily notes
and extracts anything worth keeping long-term into the appropriate folder here.

Each file should be a focused summary — not a raw log. Think of this as your curated
second brain, not a dumping ground.
