## HUB OVERVIEW

When a buyer purchases ServiceVoice AI Kit, they receive:
1. An email with a Notion workspace link (read access)
2. A ZIP file with all downloadable assets

The Notion hub is the home base — organized, clean, easy to follow.

## PAGE CONTENT BRIEFS

### 👋 Start Here
- Welcome message (Malcolm Reid voice — warm, direct)
- Quick-start checklist (3 steps: Retell setup → Make.com → go live)
- Estimated time: 60–90 minutes
- Link to deployment guide
- "Before you start" requirements list

### 📦 What's In Your Kit
- Table of all included assets with descriptions
- Download links for ZIP files (scripts, blueprints)
- Version number + last updated date

### 🚀 Deployment Guide
- Full deployment guide embedded (from `deployment-guide.md`)
- Divided into collapsible sections by step
- Screenshots for key steps (to be added)
- Video walkthrough link (once recorded)

### 📞 Call Scripts
Each niche gets its own sub-page:
- Full script embedded in Notion
- "Copy this script" instructions for Retell AI
- Customization notes (what to change, what to leave)
- Phoenix-specific callouts highlighted

### ⚙️ Make.com Workflows
- Workflow architecture overview (plain English)
- Blueprint JSON download links
- Import instructions (with screenshots)
- Variable mapping guide

### 🌎 Spanish Language Pack
*(Only visible to buyers who purchased the $97 add-on)*
- Full bilingual scripts for all 5 niches
- Spanish prompt configuration in Retell AI
- Language detection setup instructions

### 📊 Lead Tracking Sheet
- Link to Google Sheets template (make a copy)
- Column descriptions
- How to connect to Make.com

### 🆘 Troubleshooting
- Common issues table (from deployment guide)
- Retell AI known quirks
- Make.com error codes reference

### 📬 Support
- Email: ServiceVoiceAI@gmail.com
- Response time: 1 business day
- Subscription update notes (what's included in monthly updates)

## LEMON SQUEEZY DELIVERY SETUP

3. License key (if applicable)

Configure in Lemon Squeezy:
- Product type: Digital download
- Thank you page: Custom page with Notion link + next steps
- Post-purchase email: Welcome email with all links (from ConvertKit sequence Email 1)
